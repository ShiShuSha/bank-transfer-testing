# Баг-репорты

---

## Баг №1 - Некорректное округление комиссии

### ID
BUG-001

### Название
Комиссия отображается неверно при сумме 99

### Предусловие
Сервис открыт

### Шаги
1. Открыть форму перевода
2. Ввести карту
3. Ввести сумму 99

### Ожидаемый результат
Комиссия = 9

### Фактический результат
Комиссия = 0

### Severity
Major

### Priority
High

---

## Баг №2 - Нет ограничения длины номера карты

### ID
BUG-002

### Название
Поле карты принимает более 16 цифр

### Предусловие
Открыта форма ввода карты

### Шаги
1. Ввести 17 цифр

### Ожидаемый результат
Максимум 16 цифр

### Фактический результат
Возможно ввести больше 16 цифр

### Severity
Major

### Priority
Medium

---

## Баг №3 - Отрицательная сумма принимается

### ID
BUG-003

### Название
Поле суммы принимает отрицательное значение

### Предусловие
Открыта форма перевода

### Шаги
1. Ввести сумму -100

### Ожидаемый результат
Отрицательное значение запрещено

### Фактический результат
Отрицательное значение принимается

### Severity
Critical

### Priority
High
