# Тестирование банковского сервиса переводов

## Описание

В рамках задания выполнено:

- ручное тестирование банковского сервиса;
- найдено 3 дефекта;
- составлены баг-репорты;
- автоматизация Selenium + Python;
- CI-сборка намеренно падает из-за дефектов приложения.

---

## Запуск приложения

Перейти в папку dist:

```bash
cd dist
python3 -m http.server 8000
```

Открыть:
```
http://localhost:8000/?balance=30000&reserved=20001
```
